# nes-sound-talk
NES Sound Talk

## Nintendo Sound Basics

### Channels

* Square1
* Square2
* Triangle
* Noise
* Samples (not used much)

### Square1/2

* Produces square waveforms
* Can control volume
* Duty cycle (tone)

### Triangle

* Produces triangle waveforms
* no control of volume
* no control of duty cycle (tone)

### Noise

Random noise
